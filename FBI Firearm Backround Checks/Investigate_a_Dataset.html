Introduction¶

I chose the FBI Gun Data. There are two dataframes for this project, one is the nics firearm background checks dataframe(Github provied the latest version of this gun data which includes all data till 12/2019. So I downloaded this one from Github.) and another one a census dataframe. The nics firearm background checks data is from FBI's National Instant Criminal Background Check System, which provides data on the number of firearm checks by month, state, and type from 1998-11 to 2019-12; the census data includes information about population, education, income and race etc in the 50 states of the USA in sevaral years.
